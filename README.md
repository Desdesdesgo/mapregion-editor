# mapregion-editor

一定有很多同学在工作中有这样的需求：
做一个城市（或者国家）的地图，里面有多个区域，每个区域点击后有联动的数据展示。
问题在于：有些区域根本没有数据，但是有需求；
何况最近echarts 关闭了地图信息的下载入口。
因此做了这个工具，手绘地图，自动生成geojson 格式的文件供下载。
